This repository contains the source code and content for Bogdan's personal blog, focused on sharing ideas, tutorials, and notes around software development and related topics. It is structured as a real-world, living project rather than a demo, and is intended to be easy to maintain and extend over time.

The blog is designed with a focus on clarity, performance, and developer experience. The codebase aims to be a good reference for modern frontend tooling, clean architecture, and practical patterns that balance readability with flexibility for future growth.

This project may also serve as a sandbox for experimenting with new technologies, workflows, and integrations, while still keeping the main blogging experience stable for regular publishing.
